 📊 Power BI Dashboard Assignment

## Project Overview

This Power BI dashboard was developed to analyze business performance through interactive visualizations and advanced Power BI features. The report provides meaningful insights into Profit, Units Sold, and Cost of Goods Sold (COGS), helping users make data-driven decisions.

## Features

- KPI Cards for Profit, Units Sold, and COGS
- Interactive Line & Clustered Column Charts
- Waterfall Charts for variance analysis
- Pie and Donut Charts for distribution analysis
- Drill-Through functionality
- Hierarchical data exploration
- Bookmarks for report navigation
- Conditional Formatting
- Dynamic Slicers and Filters
- DAX Measures and Calculations
